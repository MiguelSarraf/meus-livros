Um minuto,
\
mísero,
\
rápido,
\
curto.
\
Custoso
\
\
Cascata fina,
\
praia colorida.
\
Acaba a areia
\
junto à rima.
\
\
A vida
\
infinitésima,
\
em tempo,
\
valerá?
\
\
Há quem só veja a sua ampulheta acabar;
\
Há quem só veja a vida passar;
\
Há quem só pense em trabalhar;
\
E acaba sem nunca ao menos, tentar rimar.
\
\
Sob a sobrevida
\
esperam,
\
eternamente.
\
\
Sob a vida
\
rio,
\
alegremente.
\
\
O que,
\
enfim,
\
importa?
