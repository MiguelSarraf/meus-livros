Estira as sinapses
\
à exaustão
\
\
Nuvem disforme
\
de palavras estranhas
\
e contas esquisitas
\
\
Uma sensação latejante
\
impregnada até os cravos,
\
até as unhas
\
\
O silencioso som
\
do rabiscar
\
das folhas de papel
\
incontáveis
\
\
Molitude lesmolisal
\
toma músculos
\
solta ossos
\
\
Retinido ressonante
\
o grito indisforme
\
do relógio rouco
\
\
Parece que o tempo
\
acaba.
\
\
E a poesia tem
\
que esperar
\
\
Mas parece que
\
a hora da poesia
\
nunca chega.
