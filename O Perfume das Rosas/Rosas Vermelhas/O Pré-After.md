A intimidade
\
à velocidade exponencial
\
deforma o tempo,
\
relativístico,
\
do passar do
\
nosso momento.
\
\
Uma bateria,
\
pulsante,
\
incessante,
\
de sangue,
\
no meu peito,
\
se vê tocar
\
um calmo
\
La vie en rose,
\
que prevê um futuro.
\
Será?
\
\
Um papo leve,
\
um assunto gostoso,
\
um olhar profundo,
\
tua pele macia.
\
\
Um papo excitante
\
Um assunto fluente
\
Um olhar, seduzente
\
Tua pele, quente.
